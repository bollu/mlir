int64_t  shift

Returns an affine map that is a translation (shift) of all result expressions in 'map' by 'shift'. Eg: input: (d0, d1)[s0] -> (d0, d1 + s0), shift = 2 returns: (d0, d1)[s0] -> (d0 + 2, d1 + s0 + 2)
